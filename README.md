
# Smart Sales & Customer Analysis Dashboard

## 📊 Project Overview

This project presents an Smart system that analyzes sales data, predicts future trends, segments customers, and provides product recommendations using Python and Tableau.

## 🚀 Features

* Sales Forecasting using Machine Learning
* Customer Segmentation (K-Means Clustering)
* Product Recommendation System
* Interactive Tableau Dashboard
* Dynamic Shipping Threshold Analysis

## 🛠️ Tools & Technologies

* Python (Pandas, Scikit-learn)
* Tableau
* Machine Learning Algorithms

## 📈 Dashboards

* Sales Performance Dashboard
* Customer Segmentation Dashboard
* Product Recommendation & Forecast Dashboard

## 🎯 Key Insights

* Identifies top-performing products
* Predicts future sales trends
* Segments customers into groups
* Highlights delivery performance (On-Time vs Late)

## 📂 Files Included

* Tableau Dashboard (.twbx)
* Dataset (.csv)
* Project Report (.pdf)
* Presentation (.pptx)
